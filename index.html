TREAT-AD AI TEP Repository

Please use a gene extensions to view the AI TEPs and note that this site is still under active development . Briefly, these TEPs were generated off of all PubMed abstracts for a given gene. ChatGPT-5.5 was used to summarize these abstracts (usually >80,000 words total) into a concise summary. That concise summary was then converted into an html webpage with a simple figure using Claude Sonnet 4.6. This process was used to minimize hallucinations that may occur when an LLM is generating de novo text and not summarizing existing text. Please use these reports at your own discretion and understand that AI generated reports are hypothesis generating and require additional validation. \n\n Detailed parameters include: \n 1) abstract pull batch size of 100 2) and abstract summarization batch size of 15.
